# Tourism-project

## Цель проекта

Всем привет! Этот проект посвящен созданию интеллектуальной системы рекомендаций индивидуальных туров на основе предпочтений и привычек туристов. Данная система позволит повысить эффективность продвижения туристических услуг и обеспечить клиентам индивидуальный подход.

## Этапы проекта
### 1. Постановка задачи и проектирование структуры данных
-  Определение состава данных, необходимых для анализа.
- Проектирование структуры базы данных с созданием таблиц (Users, Destinations, Travel History, Preferences).
### 2. Генерация и пополнение данных
- Генерация данных о пользователях с помощью библиотеки Faker.
- Загрузка дополнительного датасета с Kaggle для наполнения сведений о направлениях путешествий.
- Технологии: Python, Faker, Kaggle API, Bash.
### 3. Настройка инфраструктуры и базы данных
- Развёртывание Docker-образа с PostgreSQL для хранения данных.
- Настройка соединений и миграций данных в базу.
- Технологии: Docker, PostgreSQL.
### 4. Анализ данных и построение модели рекомендаций
- SQL-запросы для предварительного анализа данных.
- Обработка и очистка данных с помощью Python и Pandas.
- Построение механизмов рекомендаций на основе предпочтений пользователей.
- Технологии: Python, Pandas, SQL.
### 5. Визуализация и формирование итогов
- Построение дашбордов и визуализаций с помощью Яндекс DataLens.
- Документирование и подготовка отчёта о проделанной работе.
- Технологии: Python, Яндекс DataLens.
### 6. Рефакторинг и завершение проекта
- Оптимизация кода и рефакторинг проекта.
- Оформление документации и README файла.
- Тестирование всей системы и финальная шлифовка.
- Технологии: Python, GitHub.

Для сохранения датасета, найденного на kaggle.com, я решила воспользоваться API и написать bash скрипт (kaggle.sh) для автоматизации процесса. Кто смотрит это на винде, простите
https://github.com/Kaggle/kaggle-cli/tree/main

Чтобы запустить мой скрипт kaggle.sh, нужно было добавить разрешение на исполнение файла и добавить в директорию ~/.kaggle/ .json файл с моим логином и API токеном. В терминале использовала такие команды (не стала писать отдельный скрипт):

  chmod +x kaggle.sh
  cd ~/.kaggle/
  touch kaggle.json
  nano kaggle.json
